## 一、项目总览

ModbusMate 是一款 Electron 桌面端 Modbus 调试工具，面向现场工程师和调试人员。核心能力：四种功能区轮询监控、六种数据类型编解码、单点写入、线性公式换算、仪表盘大数字卡片视图、断线自动重连、设备模型与多设备并发采集、IoT 深色控制台 UI、**Modbus-RTU（USB/RS485 串口）支持**、**浏览器端调试入口（`npm run web`）**。**激活码/授权体系已彻底移除**（`scf`/`workers` 相关代码仅归档，不再恢复）。

## 六、质量保障

- **TDD 全覆盖**：每项功能先写测试确认失败，再实现代码（本轮 RTU/Web 开发由 Codex 主导、Claude 补齐收尾，均遵循此流程）
- **Mock 隔离**：service/transport 层测试使用 stub/mock 隔离真实网络与串口
- **边界防护**：空输入/Infinity/非法寄存器/超范围写入/未知区域类型/原型污染键均有校验
- **安全**：CSP 限制渲染层资源加载；Web 调试服务仅监听 `127.0.0.1` + 随机会话令牌白名单 RPC；串口 IPC 绑定可信帧；DOM 渲染统一走 `textContent`/`escapeHtml`/`escapeAttr`，避免 XSS 和选择器注入
- **无障碍**：实例弹窗对话框语义完整、键盘可达（Tab 陷阱 + Escape + 焦点归还）、图标选择器符合 WAI-ARIA radiogroup 模式
- **崩溃日志**：主进程捕获 `uncaughtException` + `unhandledRejection` 写入文件
- **中文友好**：所有错误提示面向现场工程师，不暴露技术细节
