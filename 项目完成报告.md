## 一、项目总览

ModbusMate 是一款 Electron 桌面端 Modbus-TCP 调试工具，面向现场工程师和调试人员，支持四种功能区轮询监控、六种数据类型编解码、单点写入、线性公式换算、仪表盘大数字卡片视图，以及断线自动重连。

| 指标 | 数值 |
|------|------|
| 源码行数（JS，不含 node_modules/workers/scf） | 1227 行 |
| 单元测试文件 | 4 个 |
| 测试用例数 | **46 个，全部通过** |
| 测试运行时间 | ~536ms |
| 构建产物 | `ModbusMate-0.1.0-arm64.dmg`（108MB） |
| Git 提交数 | 18 个 commit |
| 代码注释语言 | 中文 |
| 开发规范 | TDD（先写测试确认失败，再实现） |

## 二、架构分层

```
渲染层（renderer/）
├── index.html      → 界面结构（连接栏/监控栏/表格/仪表盘/日志/弹窗）
├── style.css       → 样式（深色主题/状态灯/变化闪烁/30位占位）
├── app.js          → 交互逻辑（连接/监控/写入/日志/仪表盘/配置持久化）
└── codec.js        → 纯函数编解码 + 地址换算 + 线性公式（无依赖）

桥接层
└── preload.js      → IPC 白名单（connect/disconnect/startPoll/write/saveConfig/onData/onStatus/onLog）

主进程（main/）
├── index.js           → 窗口管理 + IPC 注册 + 配置持久化 + 崩溃日志
├── modbus-service.js  → Modbus-TCP 连接/读/写（异常码中文翻译）
├── poller.js          → 轮询调度 + 写入优先队列 + 断线自动重连
└── activation.js      → 激活验证（当前已注释禁用，始终跳过）

测试（test/）
├── codec.test.js          → 编解码 31 个用例
├── modbus-service.test.js → 服务层 7 个用例（Mock 模拟 TCP）
├── poller.test.js         → 轮询调度 4 个用例
├── activation.test.js     → 激活 4 个用例（简化版）
└── simulator.js           → Modbus-TCP 模拟从站

服务端（scf/ + workers/）
├── scf/index.js        → 腾讯云 SCF 主节点激活服务
├── workers/index.js    → Cloudflare Worker 备用节点激活服务
└── scripts/generate-codes.js → 激活码生成脚本
```

## 三、功能实现清单

### 已实现功能

| 功能 | 状态 | 说明 |
|------|------|------|

| 四种区域监控 | ✅ | 保持寄存器(FC03)、输入寄存器(FC04)、线圈(FC01)、离散输入(FC02) |
| 六种数据类型 | ✅ | UInt16 / Int16 / UInt32 / Int32 / Float32(AB/BA) / Hex，每行独立设置 |
| 32 位类型占用标记 | ✅ | 自动标记下一行"由上行 32 位占用"并半透明显示 |
| 变化闪烁高亮 | ✅ | 数值变化黄色闪烁 600ms |
| 写入操作 | ✅ | 线圈 FC05 / 单寄存器 FC06 / 双寄存器 FC16，输入校验 |
| 地址模式切换 | ✅ | PLC 地址(4xxxx) 与协议地址(0 起)一键切换 |
| 线性公式转换 | ✅ | 显示值 = k × 解析值 + b，每行独立设置名称/单位/系数/偏移/小数位 |
| 仪表盘视图 | ✅ | 大数字卡片，含名称/单位/进度条，数值变化闪烁 |
| 行设置持久化 | ✅ | 类型/字序/名称/公式随监控配置自动保存恢复 |
| 断线自动重连 | ✅ | 连续 3 失败判定断线，每 5 秒重试，恢复后继续监控 |
| 配置持久化 | ✅ | 连接参数/监控配置/行设置自动保存到 userData |
| 通信日志 | ✅ | 底部日志栏，环形缓冲 200 条，时间戳 + 中文错误提示 |
| 写入优先队列 | ✅ | 写入请求插入队列头部，下个周期优先执行 |
| 只读区域保护 | ✅ | 输入寄存器/离散输入不显示写入按钮 |
| 写入公式提示 | ✅ | 设了公式的行提醒"原始值，不经公式转换" |
| 内置模拟器 | ✅ | `npm run sim` 启动，含浮点/变化值示例数据 |
| 打包构建 | ✅ | Mac DMG（108MB），Windows NSIS+便携版配置就绪 |
| 激活码系统 | 🔒 | 已注释禁用（客户端骨架 + 服务端代码 + 生成脚本完整） |

## 四、质量保障

- **TDD 全覆盖**：每项功能先写测试确认失败，再实现代码
- **真实集成测试**：`modbus-service.test.js` 对内置 Modbus-TCP 模拟从站做真实 TCP 读写验证（非 Mock）
- **边界防护**：空输入/Infinity/非法寄存器/超范围写入/未知区域类型均有校验
- **CSP 安全**：`default-src 'self'` 限制渲染层资源加载
- **崩溃日志**：主进程捕获 `uncaughtException` + `unhandledRejection` 写入文件
- **中文友好**：所有错误提示面向现场工程师，不暴露技术细节
